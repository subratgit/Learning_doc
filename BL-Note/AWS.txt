Spring boot deploy in EC2
-------------------------
create spring boot application
build and create jar file
jar file upload to S3 bucket
mark the file as public

To deploy applications  in EC2 
use putty
while createing EC2 instance you must have downloaded key pair file
by using putty key generator convert to windows relted file

open putty - use EC2 instance host name and user name
Use generated key pair file in putty ssh auth
