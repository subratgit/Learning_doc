## How to create pagination using JPA (extends with PaginationAndSortingRepository)
	- Entiy class - Class Item
	- Repository interface
		public interface ItemRepository extends PagingAndSortingRepository<Item, Long> {
			// No need to define methods for pagination and sorting explicitly
		}
	- Service class for pagination
		import org.springframework.data.domain.Page;
		import org.springframework.data.domain.PageRequest;
		public class ItemService {

			@Autowired
			private ItemRepository itemRepository;

			public Page<Item> getAllItems(int pageNo, int pageSize) {
				// Create PageRequest for pagination
				PageRequest pageable = PageRequest.of(pageNo, pageSize);

				// Fetch paginated items from repository
				return itemRepository.findAll(pageable);
			}
		}
